## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`



# 🚀 The Last Minute Saver

> **Your AI Chief of Staff for Planning, Learning, Scheduling, Navigation, and Productivity.**

The Last Minute Saver is an AI-powered productivity platform built to help students, professionals, and teams stay organized, reduce last-minute stress, and achieve long-term goals through intelligent planning.

Unlike a traditional task manager, it acts as a personal **AI Chief of Staff (Milo)** that understands your goals, prioritizes your workload, creates adaptive schedules, and continuously optimizes your day.
